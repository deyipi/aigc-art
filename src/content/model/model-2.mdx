#### Data Sources
1. **Protein structure data**: Protein Data Bank (PDB) database, including:
   - HopQ: <Link href="https://doi.org/10.2210/pdb5LP2/pdb" isExternal>PDB ID 5LP2</Link>, released 2016;
   - C1ND: <Link href="https://doi.org/10.2210/pdb4WHD/pdb" isExternal>PDB ID 4WHD</Link>, released 2015;
   - HopQ-CEACAM1 complex: <Link href="https://doi.org/10.2210/pdb6AW2/pdb" isExternal>PDB ID 6AW2</Link>, released 2018.

2. **Structure repair tool**: <Link href="https://swissmodel.expasy.org/interactive" isExternal>SWISS-MODEL server</Link>, accessed August 2025, modeling quality assessment: GMQE=0.85, QMEAN=-1.2.

3. **Simulation and analysis software**: <Link href="https://www.gromacs.org/" isExternal>GROMACS 2018.8</Link>, force field parameters from OPLS-AA/L, original literature 2001; QtGrace 0.2.6 (for data visualization); PyMOL 2.5 (for structure display).

4. **Force field and model parameters**: SPC water model and OPLS-AA/L force field parameters from official GROMACS force field library (D:/gromacs/gmx2018.8/share/gromacs/top).
